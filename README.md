# vue-myLife

### Getting Started

just clone this pro

```
git clone https://github.com/lijiahao8898/vue-myLife.git

cd vue-myLife

npm install / sudo npm install

npm start
```

### webpack

plugin：

* copy-webpack-plugin - 复制文件和目录

* html-webpack-plugin - 快速生成html

* webpack-dev-server - 开发环境的模块热替换
